Swift-Kanban Feature Update: Mar 26th, 2012
We are happy to announce a new Swift-Kanban release, ver 1.4.3 that we pushed out over the weekend with several minor yet meaningful features and improvements. Here is a snapshot of the new changes done in the version 1.4.3:-

Easily Refer Lane Policies on the Board

A simple mouse-hover on the Lane header now displays the lane description/ policy that have been defined for that lane/stage of the value stream, making it very explicit and handy to refer all the time by all team members.

Card Cloning/ Copying

If you frequently work with similar type of cards, this is the feature designed for you. The clone card option allows creation of a new card from an existing card with all the information (attributes, To-Dos,attachments etc.) copied over.

Quicker Card Deletion

If you created a wrong card on the Kanban board, deleting a card has now been made quicker with an option to delete the card from the Kanban board itself, saving any trips to the Card list view or the detailed view.

Improved Card Layout

In the zest of providing all the handy feature options right on the card, we had ended up adding just too many feature icons that made the card look a bit cluttered.We have now improved the card layout to keep minimal set of icons and merged the remaining less-frequently used feature icons under a new 'Options' icon.

Additional Filter Option in Status Reports

With the additional option to display metrics in different time-units, and the new rolled up format, the Lead/Cycle Time, etc. gets quantitative and simpler to comprehend. This will further make it easier to work upon and share the exported numerical data of the Status reports in the rolled-up time-unit you prefer.

Consolidated Export and Import of Cards

You can now export/ import the entire backlog or all the cards of various types in just a few clicks with the addition of Export/ Import options in the ‘Backlog’ and the ‘All Cards’ views. This should make the data export/ import much easier and thereby allowing better offline data changes or custom reporting.

We heartily thank all our users who directly or indirectly, provided their valuable feedback to bring out the above features.
