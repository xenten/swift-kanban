Swift-Kanban features Update
At Digite, we strongly believe in Kaizen, making continuous improvements in the way we work and the products we build. Over the last several weeks, we made numerous improvements to Swift-Kanban based on your invaluable feedback. You may have already noticed some of these changes – and we hope you liked them! This email lists some of the key enhancements that we have made.

Smart Lanes get Smarter!

  • You can Re-sequence Smart Lanes in the order you would like to see them on the Kanban board.
  • Save all the board design changes using a single ‘Save’ icon on the toolbar. You will see a prompt to ‘Save’ in case you forget, ensuring that no changes are lost by mistake.
  • You can now switch to custom Smart Lanes from attribute-based Smart Lanes, even if there are live cards on the board. Make changes to your process more easily than before!
  • The lane headers now also display the number of cards in-progress/waiting in a lane, against the WIP limit defined for that lane, for a quick view of the lane capacity utilization.

Notifications get Friendlier

  • All the notification emails now provide a link to directly navigate to the card and view more details.
  • Stay current on card status with notifications for card re-assignments/ Owner changes.
  • Get less mail by default! The default notification setting for a new user in the system is now set to ‘Daily Digest’ mode, so you get only a daily summary of all changes. Of course, you can change it any time you want.

Cards pack a greater Punch!

  • With a bigger color palette (12 colors instead of the earlier 6), you can now define and identify more card-types visually than before!
  • You can set up a To-Do list for all the Card types.
  • You can select ‘Unassigned’ as a value for Card Owner, in case you do not want to assign the card to any specific team member yet, e.g. for all cards sitting in the backlog.

Greater Insight with Metrics

  • The Histogram view for Lead/Cycle time charts is replaced with a new Lead Time distribution chart that helps you understand the typical time being taken by majority of the cards and spot the outliers better..
  • All the chart filters now have an ‘Apply’ button, so you can select a combination of filter options before applying it to the chart.

Others

  • The ‘My Projects’ widget in the Quick Links section has been changed to ‘My Recent Projects’. It now lists the last five projects you have worked on, in descending chronological order. Get to the tasks that are important quickly!
  • If you are the Administrator for Swift-Kanban in your organization, you now get access to all the projects created in the application for easier management of user accounts.
We would like to sincerely thank all Swift-Kanban users, especially those who have taken the time to regularly provide us their invaluable inputs to help us improve the product and prioritize the changes we would like to bring in. We look forward to even greater participation from the Swift-Kanban community to build the ‘right’ features in the ‘right’ way. If you have any feedback or suggestions for us, do drop us a line at support@swift-kanban.com or let us know on the Swift-Kanban Community page.
